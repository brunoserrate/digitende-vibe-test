# Landing Page - Digitende

## Objetivo Principal
Captação de leads com foco em contato via WhatsApp e formulário de contato via email.

## Funcionalidades

### 1. Formulário de Contato
- Envio automático de email via PHP ao submeter formulário
- Campos: Nome, Email, Telefone, Mensagem
- Validação de campos obrigatórios

### 2. Botão WhatsApp
- Link direto para conversa manual
- Ícone flutuante ou CTA principal

### 3. Seções da LP
- Hero com proposta de valor
- Benefícios do produto
- Prova social (depoimentos/cases)
- CTA para contato
- Footer com informações

## Tecnologias
- HTML5
- CSS3
- JavaScript puro
- PHP para envio de email
