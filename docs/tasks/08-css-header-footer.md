# Task: Implementar estilos do Header e Footer

## Descrição
Criar estilos CSS para o header (navegação) e footer da página.

## Header
- Logo alinhado
- Navegação responsiva
- Background transparente inicial
- Background sólido ao scroll
- Links estilizados
- Responsivo

## Footer
- Informações de contato
- Links úteis
- Copyright
- Background escuro
- Texto claro
- Responsivo

## Estilos Header
- Container com largura máxima
- Logo à esquerda
- Navegação à direita (desktop)
- Background transparente
- Links com hover effects
- Classe .scrolled com background sólido
- Mobile: menu simplificado

## Estilos Footer
- Background escuro
- Texto claro
- Layout em colunas
- Informações de contato
- Links úteis
- Copyright
- Responsive (1 coluna mobile, 3 colunas desktop)

## Critérios de Aceite
- [ ] Header estilizado com logo e navegação
- [ ] Header com background ao scroll
- [ ] Links com hover effects
- [ ] Footer estilizado
- [ ] Footer responsivo
- [ ] Header responsivo para mobile
