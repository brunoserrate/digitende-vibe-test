# Task: Implementar estrutura HTML semântica

## Descrição
Criar o arquivo index.html com estrutura HTML5 semântica seguindo as especificações da documentação.

## Seções a implementar
- Header com navegação e logo
- Hero Section com CTA principal
- Seção de Benefícios
- Seção de Prova Social (depoimentos/cases)
- Seção de Contato com formulário
- Footer com informações
- Botão flutuante do WhatsApp

## Requisitos
- HTML5 semântico
- Meta tags SEO básicas
- Meta tag viewport para responsividade
- Acessibilidade (aria-labels em elementos interativos)
- Links para arquivos CSS e JS

## Campos do Formulário
- Nome (text, required)
- Email (email, required)
- Telefone (tel)
- Mensagem (textarea, required)
- Botão de envio

## Links WhatsApp
- Botão na Hero Section
- Botão flutuante fixo no canto inferior direito

## Critérios de Aceite
- [ ] HTML válido e semântico
- [ ] Todas as seções implementadas
- [ ] Meta tags configuradas
- [ ] Acessibilidade implementada
- [ ] Links para CSS e JS corretos
- [ ] Links do WhatsApp configurados
