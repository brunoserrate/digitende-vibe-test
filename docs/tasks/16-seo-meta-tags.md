# Task: Implementar SEO e meta tags

## Descrição
Implementar otimização SEO básica com meta tags completas.

## Meta tags a implementar

### Essenciais
- title: Título otimizado
- description: Descrição atrativa
- viewport: Responsividade
- charset: UTF-8
- robots: index, follow

### Open Graph (Facebook/LinkedIn)
- og:title
- og:description
- og:image
- og:url
- og:type

### Twitter Cards
- twitter:card
- twitter:title
- twitter:description
- twitter:image

### Outras
- canonical: URL canônica
- theme-color: Cor da UI

## Conteúdo SEO
- Título principal (H1) com keywords
- Descrições alt em imagens
- Estrutura de headings (H1, H2, H3)
- URLs amigáveis

## Critérios de Aceite
- [ ] Meta tags essenciais implementadas
- [ ] Open Graph configurado
- [ ] Twitter Cards configurado
- [ ] Título otimizado
- [ ] Descrição otimizada
- [ ] Alt text em imagens
- [ ] Estrutura de headings correta
