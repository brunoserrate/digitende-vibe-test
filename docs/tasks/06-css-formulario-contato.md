# Task: Implementar estilos da Seção de Contato

## Descrição
Criar estilos CSS para a seção de contato com formulário.

## Requisitos
- Formulário estilizado
- Campos de input e textarea
- Labels visíveis
- Validação visual
- Botão de envio
- Responsivo

## Elementos do formulário
- Título da seção (H2)
- Nome (input type="text")
- Email (input type="email")
- Telefone (input type="tel")
- Mensagem (textarea)
- Botão de envio

## Estilos a implementar
- Container do formulário com largura máxima
- Inputs com:
  - Border radius
  - Padding
  - Border color
  - Focus state
  - Error state visual
- Textarea com altura adequada
- Labels com espaçamento
- Botão de envio estilizado
- Responsivo (largura total mobile, limitada desktop)

## Estados visuais
- Normal
- Focus
- Hover (botões)
- Error
- Success (feedback visual)

## Critérios de Aceite
- [ ] Formulário estilizado completamente
- [ ] Todos os inputs estilizados
- [ ] Focus states implementados
- [ ] Botão de envio estilizado
- [ ] Responsivo
- [ ] Estados de erro implementados
