# Task: Implementar CSS responsivo global

## Descrição
Criar arquivo css/responsive.css com todas as media queries para tornar a página totalmente responsiva.

## Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Elementos a tornar responsivos
- Container principal
- Hero Section
- Seção de Benefícios
- Seção de Contato (formulário)
- Header
- Footer
- Botão flutuante do WhatsApp
- Tamanhos de fonte
- Espaçamentos

## Media Queries

### Mobile (< 768px)
- Fontes reduzidas em 20%
- Padding reduzido
- Hero section 1 coluna
- Benefícios 1 coluna
- Formulário largura 100%
- Botão WhatsApp menor
- Header simplificado
- Footer 1 coluna

### Tablet (768px - 1024px)
- Fontes levemente reduzidas
- Padding médio
- Hero section 1 coluna
- Benefícios 2 colunas
- Formulário largura limitada
- Botão WhatsApp tamanho médio

### Desktop (> 1024px)
- Fontes completas
- Padding completo
- Hero section 1 coluna
- Benefícios 3 colunas
- Formulário largura limitada
- Botão WhatsApp tamanho completo

## Critérios de Aceite
- [ ] Media queries implementadas para todos os breakpoints
- [ ] Todos os elementos responsivos
- [ ] Testado em mobile (< 768px)
- [ ] Testado em tablet (768px - 1024px)
- [ ] Testado em desktop (> 1024px)
- [ ] Layout fluido e adaptável
